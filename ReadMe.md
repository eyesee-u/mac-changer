Mac-address changer for Linux

------installation-------

git clone https://github.com/eyesee-u/mac-changer.git

------usage-----

python3 mac_changer.py --help
 or
 python2 mac-changer.py --help
 
 -----example-----
 python3 mac-changer.py -i wlan0 -u 00:11:22:33:44:55
  or
  python3 mac-changer.py -i wlan0 -u 00:11:22:33:44:55
